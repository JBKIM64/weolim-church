## 업로드 방법

1. Supabase Dashboard → Storage → devotional-audio 버킷 선택
2. `hymns` 폴더 생성 후 위의 찬송가 mp3 파일들 업로드
3. `gospels` 폴더 생성 후 위의 복음송 mp3 파일들 업로드

업로드 완료 후 알려주시면 데이터베이스를 업데이트하겠습니다.
